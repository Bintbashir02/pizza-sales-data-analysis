# pizza-sales-data-analysis 

## Project Overview
This project analyzes pizza sales data to uncover sales trends, revenue patterns, and customer preferences using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Public dataset sourced from Maven Analytics.

## Key Insights
- Classic pizzas generated the highest revenue.
- Sales peaked during weekends and evening hours.
- A small number of pizza types contributed most of the total revenue.

## Conclusion
The analysis provides insights that can help improve inventory planning and sales strategy.
